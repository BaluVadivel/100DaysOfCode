Position

Static

This is the default position of every elements. These elements can placed to only it's default position.

Relative

Position of this element ca be changed relatives to it's default position.

OUTPUT
static
relative

Absolute

This is position is ralative to body elements position. This element is relative to it's parent element if the parent element position is other than static(default) positon.

CODE

----CSS----
.ab-container {
width: 100%;
height: 240px;
margin: 10px 0px;
background-color: aqua;
display: flex;
border: 10px solid green;
}
.ab-container > div {
width: 100px;
height: 140px;
background-color: rgba(214, 145, 54, 0.644);
border: 4px solid gray;
}
.child {
width: 80px;
height: 60px;
background-color: rgba(137, 43, 226, 0.473);
}
.relative {
position: relative;
}
.absolute {
font-size: .8rem;
position: absolute;
top: 0px;
left: 140px;
}
----HTML----
<div class="container">
<div class="parent">static
<div class="child absolute">parent : static <br> child : absolute</div>
</div>
</div>
<div class="container">
<div class="parent par-relative">relative
<div class="child absolute">parent : relative <br> child : absolute</div>
</div>
</div>

OUTPUT
static
parent : static
child : absolute
relative
parent : relative
child : absolute

Fixed

This fixed position is ralative to viewport so fixed position element is always appear on viewport it won't hide even on the page scroll.

OUTPUT

----CSS----

.fix {
width: 100px;
height: 100px;
position: fixed;
top: 0px;
right:0px;
background-color: orange;
border: 4px solid gray;
}

----HTML----

<div class="fix">position fixed top right</div>

OUTPUT

output will appear on browser's top right corner.

position fixed top right
















Sticky

This position is acts as a relative psition when it is visible in the view port. During the scroll it won's hide like relative at the edge of the viewport it act as fixed position.

CODE

----CSS----

.sti {
width: 100px;
height:40px;
position: sticky;
top: 0px;
left: calc( 50vw - 50px );
background-color: red;
border: 4px solid gray;
}

----HTML----

<div class="sti">Sticky</div>

OUTPUT
Sticky
