setTimeout()

The setTimeout() method sets a timer which executes a function or specified piece of code once the timer expires. It executes the code below the setTimeout() method until the time expiration.

Code 

console.log("Before");
setTimeout(() => {
    console.log("printing After 3 second");
}, 3000);
console.log("After");

Execution in browser console

setInterval() , clearInterval()

The setInterval() method, offered on the Window and Worker interfaces, repeatedly calls a function or executes a code snippet, with a fixed time delay between each call. It returns an interval ID which uniquely identifies the interval, so you can remove it or stops the repeating execution of later by calling clearInterval().

The clearInterval() method cancels a timed, repeating action which was previously established by a call to setInterval() using the unique interval ID which is returned by the setInterval() method.

Code - setInterval 

let intervalId = setInterval(() => {
    console.log("printing every 3 seconds");
}, 3000);

Code - clearInterval 

clearInterval(intervalId);

Execution in browser console

arrow function & this

We can not use this keyword inside the arrow function which is defined as object method. But we can use this keyword inside the normal function which is defined as object method.

We can not use this keyword inside the normal function which is called by setInterval like window method. Because this keyword inside window method pointss window object. But we can use this keyword inside the arrow function which is called by setInterval like window method. On there this keyword points it's parent object like below eample.
