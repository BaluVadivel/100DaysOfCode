Time element

<time> elelment is used to represent the specific time period. This dosen't include any style change only for beter SEO.
Here html entities used to to display < > symbols. Ref dev.w3.org

CODE:

My birthday on <time datetime="1999-09-17">September 7</time>

OUTPUT:

My birthday on September 7
