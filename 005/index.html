Form

Form is used to collect data from the user commonly using input tags.
Attributes mostly used in form element
method: GET
Collected form data is send through URL.URL length is limited to 3000 characters due to this form data lengthis also limited. Sensitive data can not used suc as password. Example: When you search something in google your searched word must be in the URL. When someone else opened the same link he will the same result.
method: POST
Send the data in HTTP or HTTPs request. Form data is not shown in the URL. Form data length or size is not limited like GET method.
action: URL
Url is a internal or external form processing unit. like "process.php"
