Day11

Cascading Style Sheet(CSS)

CSS is used to format the layout of the webpage.
Css is used to chane the font family, size, color etc. Not only fonts it controls most of the thing in the html page.

CSS can be used with html in three ways

Inline

Inline css is used to give unique style the to the single html element.
This used html elements style attribue to give style to the that element.

CODE :

<h4 style="color:red">Heading</h4>

Internal

Internal css is used to give style the to the single html page.
Internal css is defined inside <style> element which is placed inside the <head> element.

CODE : 

    <!DOCTYPE html>
    <html lang="en">
    <head>
    <style>
    h4 {
    color:red;
    }
    </style>
    </head>
    <body>
    <h4>Heading</h4>
    </body>
    </html>

External

External css is used to give style the to the many html page.
External css means linking the css stylesheet file to html page using <link> inside the <head> . Style must end with .css extension and it should not contain any html.

CODE : 

    <!DOCTYPE html>
    <html lang="en">
    <head>
    <link href="style.css" rel="stylesheet" type="text/css">
    </head>
    <body>
    <h4>Heading</h4>
    </body>
    </html>

That style.css file contain : 

    h4 {
    color:red;
    }

Width, height, scroll behaviour and dark theme

html is the root element. body is content displaying element. width 100% takes the body element as browser width. brower defaultly give some margin so that it enable horizontal scrollbar. Setting margin 0px will remove the margin and horizontal scroll bar. Giving height other than auto may restrict the content than the given height. min-height will set the minimum height for body elemenet. max-height will set the maximum heigth for the body element.
Scroll behaviour smooth will give smooth scoll winthin the page using named anchor tag navigation.
@media rule is used to apply different rules based on different media types/devices. Here i used to @media to set the theme of webpage.
learned css only dark mode from css-tricks.com

CODE :

-------css-------
html, body {
width: 100%;
margin: 0px;
min-height: 100%;
scroll-behavior: smooth;
color: black;
background-color:white;
}
@media (prefers-color-scheme: dark) {
html, body {
color: #ccc;
background-color: black;
}
h1 {
color: white;
}
}
