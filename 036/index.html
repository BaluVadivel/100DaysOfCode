Array

An array is a special variable, which can hold more than one value at a time. Array can store elements of eny type.

Each value in an array has an index and each index has a reference in a memory address. Each value can be accessed by using their indexes. The index of an array starts from zero and the last element is less by one from the length of the array.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> let data = ['Apple', 'Banana', 1, 2];
undefined
> data;
[ 'Apple', 'Banana', 1, 2 ]
> data[0]; // First item
'Apple'
> data[1]; // Last(Second) item
'Banana'
> data[2];
1
> data[3];
2

Array length property

The length property of an object which is an instance of type Array sets or returns the number of elements in that array. The value is an unsigned, 32-bit integer that is always numerically greater than the highest index in the array.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> let data = ['Apple', 'Banana', 1, 2];
undefined
> data.length
4
> data[0] // First element
'Apple'
> data[3] // Last element
2

Array Methods

Concat()

The concat() method is used to merge two or more arrays. This method does not change the existing arrays, but instead returns a new array.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> let toys = ['car','flight'];
undefined
> let foods = ['cake','biscuit'];
undefined
> toys.concat(foods); // Joins two ARRAY without converting into STRING
[ 'car', 'flight', 'cake', 'biscuit' ]
> toys + foods; // This converts ARRAY TO STRING
'car,flightcake,biscuit'

fill()

The fill() method changes all elements in an array to a static value, from a start index (default 0) to an end index (default array.length). It returns the modified array.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> let arrayOne = [1, 2, 3, 4];
undefined
> arrayOne.fill(6); // fills every index
[ 6, 6, 6, 6 ]
> arrayOne.fill(5, 1); // fills 5 from index 1 to last index
[ 6, 5, 5, 5 ]
> arrayOne.fill(0, 2, 4); // fills 0 from index 2 to index 4
[ 6, 5, 0, 0 ]

The flat() method creates a new array with all sub-array elements concatenated into it recursively up to the specified depth.

Example 

balu@balu:~$ node

Type ".help" for more information.
> let arrayOne = [0, 1, 2, [3, 4]];
undefined
> arrayOne.flat();
[ 0, 1, 2, 3, 4 ]
> let arrayTwo = [0, 1, 2, [[[3, 4]]]];
undefined
> arrayTwo.flat(2); // here 2 is depth level specifying how deep a nested array structure should be flattened
[ 0, 1, 2, [ 3, 4 ] ]

join()

The join() method creates and returns a new string by concatenating all of the elements in an array (or an array-like object), separated by commas or a specified separator string. If the array has only one item, then that item will be returned without using the separator.

slice()

The slice() method returns a shallow copy of a portion of an array into a new array object selected from start to end (end not included) where start and end represent the index of items in that array. The original array will not be modified.

SYNTAX

slice(start, end)

start [Optional]
Zero-based index at which to start extraction.

end [Optional]
Zero-based index before which to end extraction. slice extracts up to but not including end. For example, slice(1,4) extracts the second element through the fourth element (elements indexed 1, 2, and 3).

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> let animals = ['ant', 'camel', 'duck', 'cow'];
undefined
> animals.slice(1);
[ 'camel', 'duck', 'cow' ]
> animals.slice(1, 3);
[ 'camel', 'duck' ]
> animals.slice(); // Without any value returns copy of original Array
['ant', 'camel', 'duck', 'cow']

sort()

The sort() method sorts the elements of an array in place and returns the sorted array. The default sort order is ascending, built upon converting the elements into strings, then comparing their sequences of UTF-16 code units values.

The time and space complexity of the sort cannot be guaranteed as it depends on the implementation.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> let arrayMonths = ['March', 'Jan', 'Feb', 'Dec'];
undefined
> arrayMonths.sort();
[ 'Dec', 'Feb', 'Jan', 'March' ]
> let arrayNum = [1, 30, 4, 21, 100000];
undefined
> arrayNum.sort();
[ 1, 100000, 21, 30, 4 ]

splice()

The splice() method changes the contents of an array by removing or replacing existing elements and/or adding new elements in place.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> let arrayNum = [2, 4, 6, 8, 10];
undefined
> arrayNum.splice(2); // deletes every element from index 2
[ 6, 8, 10 ]
> arrayNum;
[ 2, 4 ]
> arrayNum = [2, 4, 6, 8, 10]; // returing to it's original state
[ 2, 4, 6, 8, 10 ]
> arrayNum.splice(1,2); // deletes two element from index 1
[ 4, 6 ]
> arrayNum;
[ 2, 8, 10 ]
> arrayNum = [2, 4, 6, 8, 10]; // returing to it's original state
[ 2, 4, 6, 8, 10 ]
> arrayNum.splice(1,2,100); // deletes two element from index 1 and add 100 in place.
[ 4, 6 ]
> arrayNum;
[ 2, 100, 8, 10 ]
> arrayNum = [2, 4, 6, 8, 10]; // returing to it's original state
[ 2, 4, 6, 8, 10 ]
> arrayNum.splice(1,2,100,200,300); // deletes two element from index 1 and add 100,200,300 in place.
[ 4, 6 ]
> arrayNum;
[ 2, 100, 200, 300, 8, 10 ]
