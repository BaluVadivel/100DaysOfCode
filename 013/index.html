Padding border box-sizing margin

Padding
Padding is used to give the space between content an border.
border
border is used to give borde at the edge of the container. border will increase the width and height of the container. box-sizing: border-box; will solve this issue
Margin
Margin is used to giv margin between containers
CODE :

-----CSS-------
.parent {
width: 300px;
height: 300px;
/* margin auto place the container center */
margin: 0px auto;
border: 4px solid black;
padding: 10px;
}
.child1 {
width: 100px;
height: 100px;
border: 4px solid black;
background-color: orange;
margin: 10px 0px;
}
.child2 {
width: 100px;
height: 100px;
border: 4px solid ;
/* box-sizing: border-box; this place the border inside the container so the width won't increase due to border */
box-sizing: border-box;
background-color: red;
margin: 10px 0px;
}
-----HTML------
<div class="parent">parent
<div class="child1">child1
</div>
<div class="child2">child2
</div>
</div>

OUTPUT :
parent
child1
child2
