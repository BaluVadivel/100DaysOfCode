Day 30 - started learning JavaScript

What is JavaScript ?

JavaScript is a scripting language which gives app like working functionality to websites.

Compiling js

JavaScript can be compiled in two ways

  • Compiling in browser
  • Compiling in nodejs compiler(Actually runtime)

Compiling in browser - Include JavaScript in html

<script> tag is used to include JavaScript in html.
we can directly include all JavaScript programs within the script tag. OR we can create separate filename.js file then it can be linked with html using the same <script> tag.

This <script> tag is mostly place just above the body closing </body> tag

we can Output(log or console.log) and type the JavaScript program through browser devloper tool −> web console.

Example Code

<html>
    <head>
        <title> Learning JavaScript </title>
    </head>

    <body>
        <h1> Started learning JavaScript </h1>

        <script>
            console.log("Hello World");
        </script>

        <script scr="script.js"> </script>
    </body>
</html>

Compiling in nodejs

JavaScript can be compiled in the terminal using nodejs. Before that you need to install nodejs.

we can compile the JavaScript file or else type the js program and see Output(log or console.log).

Example Code of terminal

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> console.log("Hello World");
Hello World
> "Hello World"
Hello World
OR

balu@balu:~$ node helloworld.js
Hello World
