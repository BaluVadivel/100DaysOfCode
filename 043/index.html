querySelector

The Document method querySelector() returns the first Element within the document that matches the specified selector, or group of selectors. If no matches are found, null is returned.

querySelector.html file contain below code 

<html>
<head>
    <title>querySelector</title>
</head>
<body>
    <h1>  h1 element  </h1>
    <h2 class="second">  h2 element  </h2>
    <h3 class="new">
        <a id="link" href="#">anchor tag</a>
    </h3>
    <script>
        let tagName = document.querySelector("h1");
        let className = document.querySelector(".second");
        let combination = document.querySelector("h3.new a#link[href='#']");
        console.log("querySelector tagName", tagName);
        console.log("querySelector className", className);
        console.log("querySelector combination", combination);
    </script>
</body>
</html>

querySelectorAll

The querySelectorAll() method returns all elements in the document that matches a specified CSS selector(s), as a static NodeList object. The NodeList object represents a collection of nodes. The nodes can be accessed by index numbers. The index starts at 0.

querySelectorAll.html file contain below code 

<html>
<head>
    <title>querySelectorAll</title>
</head>
<body>
    <h1 class="first">
        h1 element
    </h1>
    <h1 class="second">
        h1 element-2
    </h1>
    <h3 class="new">
        <a id="topLink" href="#">anchor tag</a>
        <a id="emptyLink" href="">anchor tag</a>
    </h3>
    <script>
        let tagName = document.querySelectorAll("h1");
        let combination = document.querySelectorAll("h3.new a[href='#']");
        console.log("querySelectorAll tagName", tagName);
        console.log("querySelectorAll combination", combination);
    </script>
</body>
</html>

getAttribute & setAttribute

The getAttribute() method returns the value of the attribute with the specified name, of an element.

The setAttribute() method adds the specified attribute to an element, and gives it the specified value. If the specified attribute already exists, only the value is changed.

getAttribute-setAttribute.html file contain below code 

<html>
<head>
    <title> getAttribute, setAttribute </title>
</head>
<body>
    <p>
        Wait 4 seconds for the change of placeholder in password input. Refresh to see again
    </p>
    <form>
        <input id="username" type="text"     placeholder="Enter User Name"> <br> <br>
        <input id="password" type="password" placeholder="Password"   >
    </form>
    <script>
        let pwd  = document.getElementById("password");
        console.log(`Password input placeholder is "${pwd.getAttribute("placeholder")}"`);
        setTimeout(
            () => {
                pwd.setAttribute("placeholder", "Enter password");
                console.log(`Password input placeholder is "${pwd.getAttribute("placeholder")}"`);
            }, 4000);
    </script>
</body>
</html>

innerHTML, textContent & innerText

The innerHTML property sets or returns the HTML content (inner HTML) of an element, including new lines which only present in raw file and not present in browser.

The textContent property sets or returns the text content of the specified node, and all its descendants,including new lines which only present in raw file and not present in browser. textContent will return the text of elements that are hidden with CSS.

The innerText property sets or returns the text content of the specified node, and all its descendants. innerText will not return the text of elements that are hidden with CSS.

innerHTML-textContent-innerText.html file contain below code 

<html>
<head>
    <title> innerHTML, textContent & innerText </title>
</head>
<body>
    <p id="first">
        first paragraph for get <b style="display: none;">bold text</b> <i>italic text</i>
    </p>
    <p>
        Wait 3 seconds, to set content for below empty paragraphs, refresh to see again
    </p>
    <p id="p1"></p>
    <p id="p2"></p>
    <p id="p3"></p>
    <script>
        let innerHTML   = document.getElementById("first").innerHTML;
        let textContent = document.getElementById("first").textContent;
        let innerText   = document.getElementById("first").innerText;
        console.log(`innerHTML get \n"${innerHTML}"`);
        console.log(`textContent get \n"${textContent}"`);
        console.log(`innerText get \n"${innerText}"`);
        setTimeout(
            () => {
                document.getElementById("p1").innerHTML ='<i>italic text</i>';
                document.getElementById("p2").textContent ='<i>italic text</i>';
                document.getElementById("p3").innerText ='<i>italic text</i>';
            }, 3000);
    </script>
</body>
</html>
