Cascading Style Sheet(CSS)

CSS is used to format the layout of the webpage.
Css is used to chane the font family, size, color etc. Not only fonts it controls most of the thing in the html page.

CSS can be used with html in three ways

Inline

Inline css is used to give unique style the to the single html element.
This used html elements style attribue to give style to the that element.

CODE :

<h4 style="color:red">Heading</h4>

Internal

Internal css is used to give style the to the single html page.
Internal css is defined inside <style> element which is placed inside the <head> element.

CODE : 

<!DOCTYPE html>
<html lang="en">
<head>
<style>
h4 {
color:red;
}
</style>
</head>
<body>
<h4>Heading</h4>
</body>
</html>

External

External css is used to give style the to the many html page.
External css means linking the css stylesheet file to html page using <link> inside the <head> . Style must end with .css extension and it should not contain any html.

CODE : 

<!DOCTYPE html>
<html lang="en">
<head>
<link href="style.css" rel="stylesheet" type="text/css">
</head>
<body>
<h4>Heading</h4>
</body>
</html>

That style.css file contain : 

h4 {
color:red;
}

Basic Selector in css

Selector is used to select the element, id and class for giving style.

element selector

body {}

Above code selects the body element.

direct child and grandchild 
body h4 {}

This selects the all the h4 elements inside the body element.

direct child only 
body > h4 {}

This selects olny the direct child of the body tag.

Class and id selector

Class selector is used to

CODE : 
<!DOCTYPE html>
<html>
<head>
<style>
.red {
color:red;
}
.blue {
color:blue;
}
#green {
color:green;
}
h5 {
color:orange;
}
</style>
</head>
<body>
<h5                              > no class & no id </h5>
<h5 class="red"                  > single class     </h5>
<h5 class="red blue"             > multiple class   </h5>
<h5 class="red blue" id="green"  > class with id    </h5>
</body>
</html>
OUTPUT :

no class & no id
single class
multiple class
class with id

Id selector has number one priority it over rides all style given by class and element name
Class selector has number two priority it over rides all style other than given by id.
Multiple classes element's last class style overrides the previous class.
Element name selector styles sre taken only whwn that element has no class and id attribute.

Background color

Foreground color
This sets the foreground color to elements

Basic Typography

Font family is used to use the specific font family from the avaliable font family.
Font weight is used to use the specific font- weight shuch as bold based on the font support
Font size is used to use the specifice font famil from the avaliable font famil.
px -> giving the font size in pixel
percentage -> This multiplies the size from its parent. when you give 200% it is 2 times the size of parent.
em -> This multiplies the size from its parent. when you give 2em it is 2 times the size of parent.
rem -> This multiplies the size from root element. when you give 2rem it is 2 times the size of root element.
rem is mostly used

CODE :

In CSS

.type
{
font-family: 'Gill Sans', 'Gill Sans MT', Calibri, 'Trebuchet MS', sans-serif;
font-size: 3rem;
font-weight: bold;
}

In HTML

<div class="type">Testing</div>

OUTPUT :

Testing
