Default parameters

Default function parameters allow named parameters to be initialized with default values if no value or undefined is passed. Default parameters are always comes at the last positions So that we can skip those parameter values while calling.

Example - with default parameters 

balu@balu-PC:~$ node
Welcome to Node.js v14.17.2.
Type ".help" for more information.
> .editor
// Entering editor mode (Ctrl+D to finish, Ctrl+C to cancel)
function multiply(a, b = 1) {
  return a * b;
}
undefined
> console.log(multiply(5, 2));
10
> console.log(multiply(5));
5

Spread syntax (...)

Spread syntax (...) allows an iterable such as an array expression or string to be expanded in places where zero or more arguments (for function calls) or elements (for array literals) are expected, or an object expression to be expanded in places where zero or more key-value pairs (for object literals) are expected.

Example - 1 

balu@balu-PC:~$ node
Welcome to Node.js v14.17.2.
Type ".help" for more information.
> let numbers = [1, 12, 3, 6];
undefined
> Math.max(1, 10, 6, 9);
10
> Math.max(numbers);
NaN
> Math.max(...numbers);
12

> console.log(...numbers);

Rest parameters

The rest parameter syntax allows a function to accept an indefinite number of arguments as an array. Rest parameter is always need to be placed at last position in the function parameters list.

Example - 1 

balu@balu-PC:~$ node
Welcome to Node.js v14.17.2.
Type ".help" for more information.
> .editor
// Entering editor mode (Ctrl+D to finish, Ctrl+C to cancel)
function sum(...theArgs) {
  return theArgs.reduce((previous, current) => previous + current);
}
undefined
> sum(1, 2, 3);
6

Example - 2 

balu@balu-PC:~$ node
Welcome to Node.js v14.17.2.
Type ".help" for more information.
> .editor
// Entering editor mode (Ctrl+D to finish, Ctrl+C to cancel)
function medals(first, ...theArgs) {
  console.log(`${first} got first rank`);
  console.log(`Participants are ${theArgs}`);
}
undefined
> medals("Rohit", "Ashok", "Raj Kumar", "Dhivagar");

Participants are Ashok,Raj Kumar,Dhivagar

Destructuring assignment

The destructuring assignment syntax is a JavaScript expression that makes it possible to unpack values from arrays, or properties from objects, into distinct variables.

Example - Destructuring arrays 

balu@balu-PC:~$ node
Welcome to Node.js v14.17.2.
Type ".help" for more information.
> .editor
// Entering editor mode (Ctrl+D to finish, Ctrl+C to cancel)
let rankList = ["Rohit", "Ashok", "Praveen", "Kumar", "Raja"];
let [ firstRank, secondRank, thirdRank ]=rankList;
undefined
> firstRank;
'Rohit'
> secondRank;
'Ashok'
> thirdRank
