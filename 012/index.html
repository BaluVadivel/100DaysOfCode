Text decoration

Underline
Gives underline to the corresponding content. This mostly indicates important word.
CODE :
<h4 style="text-decoration:underline">important</h4>
OUTPUT :

important

Overline
Gives overline to the corresponding content.
CODE :
<h4 style="text-decoration:overline">BE</h4>
OUTPUT :

BE

line through
Gives line through to the corresponding content.
CODE :
<h4 style="text-decoration:line-through">line through</h4>
OUTPUT :

line through

None
remove the text decoration of the corresponding content.
CODE :
<h4>Removing underline of a <a href="htps://www.google.com" style="text-decoration:none">link</a></h4>
OUTPUT :

Removing underline of a link

blink
This gives the blink effect. Even if the browser has compatibility we can't see any visual effect. This is a deprecated style.

Line height -> leading

Applying leading to a paragraph of text adds space in between each line which is known as line height.
default line height
CODE :
<p style="line-height: 1">First line<br>Second line</p>
OUTPUT :

First line
Second line

line height 2 is 2 times the size of font size
CODE :
<p style="line-height: 2">First line<br>Second line</p>
OUTPUT :

First line
Second line

Text vertical alignment

Vertical alignment is used to align the text in vertical manner within the line

CODE :

<p>sub <span style="font-size:26px;vertical-align: sub ;">ARE</span> you?</p>
<p>super <span style="font-size:26px;vertical-align: super ;">ARE</span> you?</p>

Basic pseudo elements

::first-line selects the first line of the element's content.
::first-letter selects the first letter of the element's content.

CODE :

-------css-------

main::first-letter {
font-size: 1.6rem;
}
main::first-line {
color: orangered;
}
------html-------

<main>
first-line selects the first line of the element's content.
<br>
first-letter selects the first letter of the element's content.
</main>

OUTPUT :

first-line selects the first line of the element's content.
first-letter selects the first letter of the element's content.

Width, height, scroll behaviour and dark theme

html is the root element. body is content displaying element. width 100% takes the body element as browser width. brower defaultly give some margin so that it enable horizontal scrollbar. Setting margin 0px will remove the margin and horizontal scroll bar. Giving height other than auto may restrict the content than the given height. min-height will set the minimum height for body elemenet. max-height will set the maximum heigth for the body element.
Scroll behaviour smooth will give smooth scoll winthin the page using named anchor tag navigation.
@media rule is used to apply different rules based on different media types/devices. Here i used to @media to set the theme of webpage.
learned css only dark mode from css-tricks.com

CODE :

-------css-------
html, body {
width: 100%;
margin: 0px;
min-height: 100%;
scroll-behavior: smooth;
color: black;
background-color:white;
}
@media (prefers-color-scheme: dark) {
html, body {
color: #ccc;
background-color: black;
}
h1 {
color: white;
}
}
