The HTML lang attribute is used to identify the language 
of text content on the web. This information helps search 
engines return language specific results, and it is also used 
by screen readers that switch language profiles to provide 
the correct accent and pronunciation.
The lang attribute is forgotten surprisingly often, perhaps 
because it makes no apparent difference unless you use a 
screen reader or you are a search engine.
