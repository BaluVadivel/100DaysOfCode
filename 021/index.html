Flex Wrap

nowrap (default) - this won't wrap the element even if the elements can't fit in single row or column. It reduce the child elements width to fit within single row or column.
wrap - This wrap the elements when it can't fit in single row or column. It won't change child elements width to fit in single row or column.
wrap-reverse - This wrap the elements in reverse order when it can't fit in single row or column. It won't change child elements width to fit in single row or column.

CODE

----CSS----
.flex-wrap div {
    width: 80vw;
}
.flex-wrap-wrap {
    flex-wrap: wrap;
}
.flex-wrap-wrap-reverse {
    flex-wrap: wrap-reverse;
}
----HTML----
<span>flex-wrap: nowrap (default)</span>
<nav class="container flex-wrap flex-wrap-nowrap">
    <div>Home</div>
    <div>Search</div>
    <div>Logout</div>
</nav>
<span>flex-wrap: wrap</span>
<nav class="container flex-wrap flex-wrap-wrap">
    <div>Home</div>
    <div>Search</div>
    <div>Logout</div>
</nav>
<span>flex-wrap: wrap-reverse</span>
<nav class="container flex-wrap flex-wrap-wrap-reverse">
    <div>Home</div>
    <div>Search</div>
    <div>Logout</div>
</nav>

Logout

Flex grow shrink basis

flexis the short hand property for three values
SYNTAX
flex: [flex-grow] [flex-shrink] [flex-basis] ;

Flex grow when it's parent size increases it will grow comparing to it's sibling grow value.
Flex shrink when it's parent size decreases it will shrink comparing to it's sibling shrink value.
Flex basis this sets minimum flex value.

CODE

----CSS----
.flex > .home {
    /* flex: 1 0 100px */
    flex-grow: 1;
    flex-shrink: 0;
    flex-basis: 100px;
}
.flex > .search {
    /* flex: 0 1 100px */
    flex-grow: 0;
    flex-shrink: 1;
    flex-basis: 100px;
}
.flex > .logout {
    /* flex: 1.5 2 160px */
    flex-grow: 1.5;
    flex-shrink: 2;
    flex-basis: 160px;
}
----HTML----
<nav class="container flex">
    <div class="home">Home</div>
    <div class="search">Search</div>
    <div class="logout">Logout</div>
</nav>

OUTPUT

Home
Search
Logout
