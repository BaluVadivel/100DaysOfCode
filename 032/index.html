Comments

In javaScript comments can be used to explain JavaScript code, and to make it more readable. Comments can also be used to prevent execution, when testing alternative code.

Single line Comments

Single line comments start with //. Any text between // and the end of the line will be ignored by JavaScript (will not be executed).

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> let a = 10; //Declaring variable
undefined
> a
10

Multi line Comments

Multi-line comments start with /* and end with */. Any text between /* and */ will be ignored by JavaScript.

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> let a = 10; //Declaring variable
undefined
> .editor
// Entering editor mode (^D to finish, ^C to cancel)
let b = 20;
/*
multi line
comment
*/
undefined
> b
20

String

The String object is used to represent and manipulate a sequence of characters. String can be any text inside single or double quotes. The backslash (\) escape character turns special characters into string characters:

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> let nameOne = "Hi i'm balu";
undefined
> nameOne
"Hi i'm balu"
> let nameTwo = 'Hi i\'m balu';
undefined
> nameTwo
"Hi i'm balu"
> let nameThree = 'He is called "Balu"';
undefined
> nameThree
'He is called "Balu"'
> let nameFour = "He is called \"Balu\"";
undefined
> nameFour
"He is called "Balu""

String index

The characters in a string can be located using their index value, which represents their position in the string. Enclosing the index in square brackets will select a single character from a string: Characters are indexed from left to right, starting from the position 0, not 1! Blank spaces are also counted as characters.
Last char index is one less than it's length because index starts from 0.

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> let stringName = "Balu Vadivel";
undefined
> stringName
'Balu Vadivel'
> stringName[0]
'B'
> stringName[5]
'V'
> stringName[4] // index 4 is blank space
' '
> stringName[20]
undefined

charAt()

The String object's charAt() method returns a new string consisting of the single UTF-16 code unit located at the specified offset into the string.

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> let myName = "Balu Vadivel";
undefined
> myName.charAt(0);
'B'
> myName.charAt(5);
'V'
> myName.charAt(99); // Returns empty string
''

charCodeAt()

The charCodeAt() method returns an integer between 0 and 65535 representing the UTF-16 code unit at the given index.

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> let myName = "Balu Vadivel";
undefined
> myName.charCodeAt(0); //returns 66, the Unicode value for B
66
> myName.charCodeAt(1); //returns 97, the Unicode value for a
97
> myName.charCodeAt(20); //returns NaN, index is out of range
NaN

concat()

The concat() method concatenates the string arguments to the calling string and returns a new string.

Example

balu@balu:~$ node

Type ".help" for more information.
> let f = "Balu", l = "Vadivel";
undefined
> let name = f.concat(' ',l);
undefined
> name
'Balu Vadivel'

undefined
> Name
'Balu Vadivel'

'Balu VadivelBalu Vadivel'

endsWith()

The endsWith() method determines whether a string ends with the characters of a specified string, returning true or false as appropriate. Optionally we can give string length value, that position considered as end of the string.

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
let Name = "Balu Vadivel"
> Name.endsWith('vel');
true
> Name.endsWith('ve');
false
> Name.endsWith('lu',4); // 4 is string length, not index
true

includes()

The includes() method determines whether one string may be found within another string, returning true or false as appropriate.
The position within the string at which to begin searching for searchString. Here value is index value(Defaults to 0).

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> let string = "Hi i'm Balu";
undefined
> string
"Hi i'm Balu"
> string.includes("Balu");
true
> string.includes("balu"); // case sensitive
false
> string.includes("Hi",0); // first character index value 0
true
> string.includes("Hi",1);
false

indexOf()

The indexOf() method returns the index within the calling String object of the first occurrence of the specified value, starting the search at fromIndex. Returns -1 if the value is not found. The index of the first occurrence of searchValue, or -1 if not found.
An integer representing the index at which to start the search. Defaults to 0.

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> let string = "Hi i am Balu";
undefined
> string
'Hi i am Balu'
> string.indexOf("H");
0
> string.indexOf("h"); // case sensitive
-1
> string.indexOf("i",0);
1
> string.indexOf("i",2); // start search from index 2
3
> string.indexOf("s",2); // searchValue not found
-1
> string.indexOf("i",20);
-1
> string.indexOf(""); // searching an empty string
0
> string.indexOf("",4); // searching an empty string from index 4
4
> string.indexOf("",12); // searching an empty string from last index 12
12
> string.indexOf("",20); // searching an empty string from out of range index value
12

lastIndexOf()

The lastIndexOf() method returns the index within the calling String object of the last occurrence of the specified value, searching backwards from fromIndex (The default value is +Infinity). Returns -1 if the value is not found.

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> let string = "Hi i am Balu";
undefined
> string;
'Hi i am Balu'
> string.lastIndexOf("a");
9
> string.lastIndexOf("a",8); // search backwards from index 8
5
> string.lastIndexOf(""); // search empty string
12
> string.lastIndexOf("",7);
7
> string.lastIndexOf("",20);
12

localeCompare()

The localeCompare() method returns a number indicating whether a reference string comes before, or after, or is the same as the given string in sort order.

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> 'a'.localeCompare('b');// The letter "a" is before "b" yielding a negative value
-1
> 'b'.localeCompare('a');// The letter "b" is before "a" yielding a positive value
1
> 'a'.localeCompare('a');
0

padEnd()

The padEnd() method pads the current string with a given string (repeated, if needed) so that the resulting string reaches a given length. The padding is applied from the end of the current string.

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> let strOne = 'Balu Vadivel';
undefined
> strOne.padEnd(16,'123');
'Balu Vadivel1231'
> strOne.padEnd(16); // adds empty spaces
'Balu Vadivel    '
> strOne.padEnd(4); // can't reduce string length
'Balu Vadivel'

padStart()

The padStart() method pads the current string with another string (multiple times, if needed) until the resulting string reaches the given length. The padding is applied from the start of the current string.

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> let strOne = 'Balu Vadivel';
undefined
> strOne.padStart(16,'123');
'1231Balu Vadivel'
> strOne.padStart(16); // adds empty spaces
'    Balu Vadivel'

repeat()

The repeat() method constructs and returns a new string which contains the specified number of copies of the string on which it was called, concatenated together.

Example

    balu@balu:~$ node
    Welcome to Node.js v12.20.1.
    Type ".help" for more information.
    > let strOne = 'happy+';
    undefined
    > strOne.repeat(3);
    'happy+happy+happy+'
    > strOne.repeat(0); // empty String
    ''
    > strOne.repeat(2.5); // count will be converted to integer
    'happy+happy+'
    > strOne.repeat(-3); // Negative value error
    Uncaught RangeError: Invalid count value at String.repeat (<anonymous>)

replace()

The replace() method returns a new string with some or all matches of a pattern replaced by a replacement. The pattern can be a string or a RegExp, and the replacement can be a string or a function to be called for each match. If pattern is a string, only the first occurrence will be replaced.

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> let strOne = 'The fox jumps over the lazy dog. The dog was not yet reacted.';
undefined
> strOne.replace('dog','sheep');
'The fox jumps over the lazy sheep. The dog was not yet reacted.'

replaceAll()

Example
