Conditional statements - decision making

Conditional statements are used to skip or execute group(block) of statement based on the result of some condition.

if...else

The if statement executes a statement if a specified condition is truthy. If the condition is falsy, another statement can be executed.

if-else.js contains below code

let numOne = 6;
if ( numOne%2 === 0 ) {
    console.log(`numOne holds ${numOne} which is a EVEN number`);
} else {
    console.log(`numOne holds ${numOne} which is a ODD number`);
}

numOne = 7;
if ( numOne%2 === 0 ) {
    console.log(`numOne holds ${numOne} which is a EVEN number`);
} else {
    console.log(`numOne holds ${numOne} which is a ODD number`);
}

Compiling if-else.js in node

balu@balu:~$ node if-else.js

numOne holds 7 which is a ODD number

else if

else if atatement is used to take multiway decision. This statement is formed by joining if....else statement in which each else contains another if...else.

else-if.js contains below code

let numOne = 6;
if ( numOne > 0 ) {
    console.log(`numOne holds ${numOne} which is a POSITIVE number`);
} else if ( numOne < 0 ) {
    console.log(`numOne holds ${numOne} which is a NEGATIVE number`);
} else {
    console.log(`numOne holds ${numOne} which is neither POSITIVE or NEGATIVE`);
}

numOne = -6;
if ( numOne > 0 ) {
    console.log(`numOne holds ${numOne} which is a POSITIVE number`);
} else if ( numOne < 0 ) {
    console.log(`numOne holds ${numOne} which is a NEGATIVE number`);
} else {
    console.log(`numOne holds ${numOne} which is neither POSITIVE or NEGATIVE`);
}

numOne = 0;
if ( numOne > 0 ) {
    console.log(`numOne holds ${numOne} which is a POSITIVE number`);
} else if ( numOne < 0 ) {
    console.log(`numOne holds ${numOne} which is a NEGATIVE number`);
} else {
    console.log(`numOne holds ${numOne} which is neither POSITIVE or NEGATIVE`);
}

Nested if...else

This statement is formed by joining if...else statements either in the if block or in the else block or both.

nested-if-else.js contains below code

let numOne = 6, numTwo = -6, numThree = 16;
if ( numOne > numTwo ) {
    if (numOne > numThree) {
        console.log(`numOne holds ${numOne} which is the greater than both numTwo &\ numThree`);
    } else {
        console.log(`numThree holds ${numThree} which is the greater than both numOne &\ numTwo`);
    }
} else {
    if ( numTwo > numThree) {
        console.log(`numTwo holds ${numTwo} which is the greater than both numOne &\ numThree`);
    } else {
        console.log(`numThree holds ${numThree} which is the greater than both numOne &\ numTwo`);
    }
}

Compiling nested-if-else.js in node

balu@balu:~$ node nested-if-else.js

switch

The switch statement evaluates an expression, matching the expression's value to a case clause, and executes statements associated with that case, as well as statements in cases that follow the matching case.

The optional break statement associated with each case label ensures that the program breaks out of switch once the matched statement is executed

switch.js contains below code

let expr = 'monday';
switch (expr) {
    case 'monday':
    case 'tuesday':
    case 'wednesday':
    case 'thursday':
    case 'friday':
      console.log('Monday to friday');
        break;
    case 'saturday':
    case 'sunday':
      console.log('Holiday');
      break;
    default:
      console.log(`Sorry, there is no ${expr} in a week`);
}

expr = 'memesday';

}
