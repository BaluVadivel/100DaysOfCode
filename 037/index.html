Loop

Loop is used to execute a block of code repeatedly for a number of times based on a condition.

The statements for loops provided in JavaScript are:

  • while statement
  • do...while statement
  • for statement
  • break statement
  • continue statement
  • labeled statement
  • for...in statement
  • for...of statement

while

The while statement creates a loop that executes a specified statement as long as the test condition evaluates to true. The condition is evaluated before executing the statement.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> let n = 0, result = '';
undefined
> while(n<5) { ++n; result=result+n }
'12345'
> result
'12345'
> n
5
> while(n<5) { ++n; result=result+n } // Loop condition false at first cycle so block code not executed
undefined

do...while

The do...while statement creates a loop that executes a specified statement until the test condition evaluates to false. The condition is evaluated after executing the statement, resulting in the specified statement executing at least once.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> let n = 0, result = '';
undefined
> do { ++n; result=result+n } while(n<5);
'12345'
> result
'12345'
> n
5
> do { ++n; result=result+n } while(n<5); // Loop condition false at first cycle even though block code executed once
'123456'

for

The for statement creates a loop that consists of three optional expressions, enclosed in parentheses and separated by semicolons, followed by a statement (usually a block statement) to be executed in the loop.

SYNTAX

for ([initialization]; [condition]; [final-expression]) { statement }

initialization
Declare new variables or intialize value for already declared variables.

condition
An expression to be evaluated before each loop iteration.

final-expression
Update or increment the counter variable.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> let str = '';
undefined
> for (let i = 0; i < 5; i++) { str = str + i; }
'01234'
> str
'01234'

break

The break statement terminates the current loop, switch, or label statement and transfers program control to the statement following the terminated statement.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> .editor
// Entering editor mode (Ctrl+D to finish, Ctrl+C to cancel)
let i = 0;
while (i < 6) {
  if (i === 3) {
    break;
  }
  i = i + 1;
}
console.log(i); // Below is OUTPUT
3

continue

The continue statement terminates execution of the statements in the current iteration of the current or labeled loop, and continues execution of the loop with the next iteration.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> .editor
// Entering editor mode (Ctrl+D to finish, Ctrl+C to cancel)
let text = '';
for (let i = 0; i < 6; i++) {
  if (i === 3) {
    continue;
  }
  text = text + i;
}
console.log(text); // Below is OUTPUT
01245

Label

You can use a label to identify a loop, and then use the break or continue statements to indicate whether a program should interrupt the loop or continue its execution.
Note that JavaScript has no goto statement, you can only use labels with break or continue.

Example - CONTINUE LABEL 

balu@balu:~$ node
Welcome to Node.js v14.17.2.
Type ".help" for more information.
> .editor // CONTINUE LABEL
// Entering editor mode (Ctrl+D to finish, Ctrl+C to cancel)
let A, B;
console.log("<----OUTPUT---->");
loop1:
for (A = 0; A < 3; A++) {
   loop2: // Here this lable just identifies child loop
   for (B = 0; B < 3; B++) {
      if (A === 1 && B === 1) {
         continue loop1;
      }
      console.log('A = ' + A + ', B = ' + B);
   }
}
<----OUTPUT---->

A = 0, B = 1
A = 0, B = 2

A = 2, B = 1

for...of

The for...of statement creates a loop iterating over iterable objects, including: built-in String, Array, array-like objects (e.g., arguments or NodeList), TypedArray, Map, Set, and user-defined iterables. It invokes a custom iteration hook with statements to be executed for the value of each distinct property of the object.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> .editor
// Entering editor mode (Ctrl+D to finish, Ctrl+C to cancel)
let array = ['a', 'b', 'c'];
    console.log("<----OUTPUT---->");
for (const element of array) 
{
  console.log(element);
}
<----OUTPUT---->
a
b
c
undefined

for...in

The for...in statement iterates over all enumerable properties of an object that are keyed by strings (ignoring ones keyed by Symbols), including inherited enumerable properties. This statement use object. Refer object concept.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> .editor
// Entering editor mode (Ctrl+D to finish, Ctrl+C to cancel)
let object = { a: 1, b: "two", c: 3 };
    console.log("<----OUTPUT---->");
for (const property in object)
{
    console.log(`${property} = ${object[property]}`);
}
<----OUTPUT---->
a = 1
b = two
c = 3
undefine
