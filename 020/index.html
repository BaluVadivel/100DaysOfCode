Filter

The filter CSS property applies graphical effects like blur or color shift to an element including it's border

CODE

----CSS----

img, .multi {
width: 80vw;
max-width: 240px;
height: 80vw;
max-height: 240px;
border: 4px solid orange;
}
.multi {
font-size: 3rem;
filter: blur(2px) brightness(2);
}
.blur {
filter: blur(2px);
}
.brightness {
filter: brightness(.5);
/* 50% brightness */
}
.contrast {
filter:contrast(2)
}
.drop-shadow {
filter: drop-shadow(6px 6px 4px red);
}
.grayscale {
filter: grayscale(1);
}
.hue-rotate {
filter: hue-rotate(90deg);
}
.invert {
filter: invert(1);
}
.opacity {
filter: opacity(.2);
}
.saturate {
filter: saturate(6);
}
.sepia {
filter: sepia(2);
}

Backdrop-Filter

Backdrop-Filter only applies effects to the background.

CODE

----CSS----

.backdrop {
width: 80vw;
max-width: 240px;
height: 80vw;
max-height: 240px;
font-size: 3rem;
background-color: rgba(128, 255, 0, 0.747);
-webkit-backdrop-filter: invert(100%);
backdrop-filter: invert(100%);
}
----HTML----

<div class="backdrop">
Back drop
</div>

Backdrop-Filter is not widwly supported.caniuse

OUTPUT

Back drop

Break

Break After - changes page break after an element.
Break Before - changes page break before an element.
Break inside - changes page break inside an element.

CODE

OUTPUT

break-after break-before
Avoid a page/column/region break inside the element
