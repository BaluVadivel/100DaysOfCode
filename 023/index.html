ITEMS Align Justify Place

These three properties has multiple values here all the output used only center value.
place items is the shortthand property to use both align and justify.
Syntax : place-items: [align-value] [justify-value]

The values are

  • start(flex-start)
  • end(flex-end)
  • center
  • stretch(default)
  • baseline

CODE

----CSS----
.items {
    grid-template-columns: repeat(2,1fr);
}
.items-align {
    align-items: center;
}
.items-justify {
    justify-items: center;

Grid row & column line naming

Consider an grid container which has four items placed in 2 rows and 2 columns.
That container will have three row line and three column line.
These lines has default names

  • Three rows lines has name 1,2,3 respectively.
  • Three columns lines has name 1,2,3 respectively.
We can give prefered name to those lines using grid-template-rows, grid-template-columns, grid-template, grid.

Example 


grid-template-rows    : [row-start] 30px [row-2] auto [row-3] 30px [row-end] ;



grid-template-columns : [col-start] 1fr  [col-2] 1fr  [col-3] 1fr  [col-end] ;

Grid column, start, end & row, start, end , area

grid area is the shorthand property for col start,end row start,end.

Syntax
grid-area: [row-start] / [col-start] / [row-end] / [col-end] ;

CODE

    ----CSS----

.grid-lines {
    grid-template: [row-start] 30px [row-2] auto [row-3] 30px [row-end] / [col-start] 1fr  [col-2] 1fr  [col-3] 1fr  [col-end] ;
}
.h {
    /*
    grid-column-start: 1;
    grid-column-end: 4;
    grid-row-start: 1;
    grid-row-end: 2;
    OR 
    
    grid-column-start: col-start;
    grid-column-end: col-end;
    grid-row-start: row-start;
    grid-row-end: row-2;
    
    OR
    
    grid-column-start: col-start;
    grid-column-end: span 3;
    above span value take three column line after col-start.
    OR
    grid-column: col-start / col-end;
    grid-row: row-start / row-2;
    OR
    More simple below grid-area
    it's syntax
    grid-area: [row-start] / [column-start] / [row-end] / [column-end] ;
    */
    grid-area: 1 / 1 / 3 / 3 ;
}
.m {
    grid-area: 2 / 1 / 3 / 3 ;
}
.s {
    grid-area: 2 / 3 / 3 / 4 ;
}

----HTML----

<div class="grid container grid-lines">
    <div class="h">header</div>
    <div class="m">main</div>
    <div class="s">side</div>
    <div class="f">footer</div>
</div>
