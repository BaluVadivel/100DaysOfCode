DIV & SPAN TAGS

Div tag is used to group block level element. Span tag is used to group inline level element. This tag has both opening and closing tags

CODE :

<div>this is div</div>
<span>this is span</span>
<div>this is div</div>

OUTPUT :

this is div
this is span
this is div
