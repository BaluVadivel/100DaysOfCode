Display inline,block and inline-block

inline
inline element heights are automatically taken as same as line height. We can't change it's height.
block
block level elements always appear in new line
inline-block
inline-block element acts as an inline element also we change set the size.
CODE :

---CSS---
ul {
width: 100%;
margin:0px;
padding:0px;
height: 80px;
background-color: orange;
}
li {
display:inline-block;
height: 100%;
background-color: black;
line-height: 80px;
font-size: 1.4rem;
margin: 0 20px;
padding: 0 22px;
}
a {
text-decoration:none;
color: red;
}
li:hover {
background-color:#ddd;
}
---HTML---
<ul>
<li><a href="#">HOME</a></li>
<li><a href="#">ABOUT</a></li>
<li><a href="#">SKILL</a></li>
<li><a href="#">CONTACT</a></li>
</ul>

OUTPUT :

  • HOME
  • ABOUT
  • SKILL
  • CONTACT

Box shadow

CODE :

---CSS---
div {
display:inline-block;
width: 80px;
height: 80px;
border: 3px solid black;
margin: 20px;
}
#bottom-right {
box-shadow: 6px 6px red;
}
#top-left {
box-shadow: -6px -6px red;
}
#top-left-blur {
box-shadow: -6px -6px 6px red;
}
#top-left-blur-spread {
box-shadow: -6px -6px 6px 6px red;
}
#outer-blur {
box-shadow: 0 0 4px red;
/*Blur always gives a minor spread. If we need more spread we can add 4th value as spread */
}
#inner-blur {
box-shadow: inset 0 0 4px red;
/*Blur always gives a minor spread. If we need more spread we can add 4th value as spread */
}
---HTML---
<div id="bottom-right">bottom-right</div>
<div id="top-left">top-left</div>
<div id="top-left-blur">top-left-blur</div>
<div id="top-left-blur-spread">top-left-blur-spread</div>
<div id="outer-blur">outer-blur / spread</div>
<div id="inner-blur">inner-blur / spread</div>

OUTPUT :

bottom-right
top-left
top-left-blur
top-left-blur-spread
outer-blur / spread
inner-blur / spread
