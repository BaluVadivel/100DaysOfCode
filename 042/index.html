DOM

The Document Object Model (DOM) connects web pages to scripts or programming languages by representing the structure of a document—such as the HTML representing a web page—in memory. Usually it refers to JavaScript, even though modeling HTML, SVG, or XML documents as objects are not part of the core JavaScript language.

The DOM represents a document with a logical tree.

Selecting Element

We need to select element before using or manipulating them. we can select element using below document methods.

getElementById

The getElementById() method returns one element that has the ID attribute with the specified value. This method is one of the most common methods in the HTML DOM, and is used almost every time you want to manipulate, or get info from, an element on your document.

Link contain below code 

<html>
<head>
    <title>getElementById</title>
</head>
<body>
    <h1>  h1 element  </h1>
    <h2>  h2 element  </h2>
    <h3 id="new">  h3 element  </h3>
    <script>
        let idElement = document.getElementById("new");
        console.log(idElement);
        console.log('Above element got id attribut value "new"');
    </script>
</body>
</html>

Browser output and console

getElementsByTagName

The getElementsByTagName() method returns a collection of all elements in the document with the specified tag name, as an HTMLCollection object. The HTMLCollection object represents a collection of nodes. The nodes can be accessed by index numbers. The index starts at 0.

Link contain below code 

<html>
<head>
    <title>getElementsByTagName</title>
</head>
<body>
    <h1>  h1 element  </h1>
    <h2>  h2 element  </h2>
    <h1 class="new">  h1 element with class "new"  </h1>
    <script>
        let h1Element = document.getElementsByTagName("h1");
        console.log(h1Element);
        console.log(`There are ${h1Element.length} h1 element present.`);
    </script>
</body>
</html>

Browser output and console

getElementsByClassName

The getElementsByClassName() method returns a collection of all elements in the document with the specified class name, as an HTMLCollection object. The HTMLCollection object represents a collection of nodes. The nodes can be accessed by index numbers. The index starts at 0.

Link contain below code 

<html>
<head>
    <title>getElementsByClassName</title>
</head>
<body>
    <h1 class="first">  h1 element  </h1>
    <h1 class="new">  h1 element with class "new"  </h1>
    <p class="new">  p element with class "new"  </p>
    <script>
        let newClassElement = document.getElementsByClassName("new");
        console.log(newClassElement);
        console.log(`There are ${newClassElement.length} "new" class element present.`);
    </script>
</body>
</html>
