Basic math operations - Arithematic operators

+
Addition to add two numbers
-
Subtraction to subtract one number from another number
*
Multiplication to multiply two numbers
/
Divides the left number by the right
%
Remainder(modulo) to return remainder left after integer division
**
Exponent(power) that is left side number is multiplied by itself right side number no of times.
example: 3 ** 2 ⇒ 32

Naming Variable - Identifier

Identifier represents variables, functions and properties. Variable can contain alphabets, digits, $, _. But may not start with digit. In javaScript Identifiers are camelCase stylized.
Use captial letter in each word's first letter expect first word's first letter.
Use currentDate instead of current_date or currentdate
Use isLoggedIn instead of is_logged_in or isloggedin

Example

balu@balu:~$ node
Welcome to Node.js v12.20.1.
Type ".help" for more information.
> let currentDate = 7;
undefined
> currentDate;
7
