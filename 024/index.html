Day 24

CSS selector

Basic Selector

Universal selector

This selects all the elements expect ::after & ::before (Pseudo Element) .

CODE

            * {
                border-radius: 4px;
            }

Name or type selector

Select the element using element name. Below code selects every div element inside body element.

CODE

            div {
                border-radius: 4px;
            }

Class selector

Selecting element using it's classname.

CODE

            .output {
                border-radius: 4px;
            }

Id selector

Selecting element using it's unique Id name.

CODE

            #output {
                border-radius: 4px;
            }

Combinators

Descendant Selector

Select elements that are descendants of the first element.
Select every elements that are direct child , grand child nad multi level grand child of the first element.

CODE

            div a {
                background-color: yellow;
            }

Direct Child Selector

Select every elements that are direct child of the first element.

CODE

            div > a {
                border-radius: 4px;
            }

Previous Sibling Selector

There is no Previous Sibling Selector avaliable in CSS till CSS3.

General Sibling or Succeeding Sibling Selector

Select elements that are siblings of the first element and come after the first element.

CODE

            div ~ a {
                border-radius: 4px;
            }

Adjacent Sibling Selector

Select elements that are siblings of the first element and come directly (Immediately) after the first element.

CODE

            div + a {
                border-radius: 4px;
            }

OR Selector or List Selector

Select elements that match any selector in the list.
We can give same style to multiple elements using or selector.
Seleting multiple elements and giving same style to all.

CODE

            div, a {
                border-radius: 4px;
            }

And Selector

Select elements that match all the selector combinations.
Below code selects div element which has box class.

CODE

            div.box {
                border-radius: 4px;
            }

Attribute Selector

Has Attribute Selector

Select elements that have that attribute.
Below code selects all the elements which contains href attribute

CODE

            [href] {
                color: yellow;
            }

Selected tags

<a href="#"> </a>
<a href="/contact"> </a>

Non Selected tags

<a> no link </a>

Exact Attribute Selector

Select elements that have that attribute with exactly that value.

CODE

            [class="box"] {
                color: red;
            }

Selected tags

<div class="box"> </div>
<span class="box"> </span>

Non Selected tags

<div class="box price"> </div>
<div class="box-price"> </div>

Exact Attribute with whitespace

Represents elements with an attribute name of attr whose value is a whitespace-separated list of words, one of which is exactly value.

CODE

            [class~="box"] {
                color: red;
            }

Selected tags

<div class="box"> </div>
<div class="box price"> </div>
<div class="abc box price"> </div>

Non Selected tags

<div class="box-price"> </div>

Exact Attribute with hyphen

Represents elements with an attribute name of attr whose value can be exactly value or can begin with value immediately followed by a hyphen, - (U+002D). It is often used for language subcode matches.

CODE

            [class|="box"] {
                color: red;
            }

Selected tags

<a class="box">LINK</a>
<a class="box-price">LINK</a>

Non Selected tags

<a class="box price">LINK</a>
<a class="price box-b">LINK</a>
<a class="price-box-b">LINK</a>

Begins With Attribute

Select every elements that have that attribute which start with that value.

CODE

            [href^="http"] {
                color: red;
            }

Selected tags

<a href="https://balu.dev">LINK</a>
<a href="http://balu.dev">LINK</a>
<a href="http-def">LINK</a>

Non Selected tags

<a href="HTTPS://balu.dev">LINK</a>
<a href="HTTP://balu.dev">LINK</a>
<a href="HTTP-def">LINK</a>

Ends With Attribute

Select elements that have that attribute which end with that value.

CODE

            [href$="pdf"] {
                color: red;
            }

Selected tags

<a href="pdf">LINK</a>
<a href=".pdf">LINK</a>
<a href="abcpdf">LINK</a>

Non Selected tags

<a href="PDF">LINK</a>
<a href=".PDF">LINK</a>

Substring Attribute

Select elements that have that attribute which contain that.

CODE

            [href*="pdf"] {
                color: red;
            }

Selected tags

<a href="start-pdf-end">LINK</a>
<a href="startpdfend">LINK</a>

Non Selected tags

<a href="start-PDF-end">LINK</a>
<a href="startPDFend">LINK</a>

Case insensitive

Syntax:
[attr operator="value" i]
Adding an i (or I) before the closing bracket causes the value to be compared case-insensitively (for characters within the ASCII range).

CODE

            [href*="pdf"] {
                color: red;
            }

Selected tags

<a href="start-PDF-end">LINK</a>
<a href="startPDFend">LINK</a>
<a href="start-pdf-end">LINK</a>
<a href="startpdfend">LINK</a>
