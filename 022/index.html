Container Properties

grid-template-columns

grid-template-columns is used to give each column width.
The number of column is taken from number of width values given.

auto value will take entire remaining width.
fr (fraction) value will divide entire remaining width and give it to each column.

grid-template-columns: 1fr 2fr; this will give one fraction for first column , two fraction for second column from the entire remaining width.

CODE

----CSS----

.grid-template-columns {
    grid-template-columns: 100px 70px 40px auto;
    /*
    grid-template-columns: repeat(4, 1fr); = grid-template-columns: 1fr 1fr 1fr 1fr;
    We can also set pixel percentage values
    */
}

----HTML----

<div class="grid container grid-template-columns">
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
    <div>5</div>
    <div>6</div>
</div>

auto fill in minmax

This always give remaining space in the specified width.
Fills our container with extra column.

CODE

----CSS----

.grid-template-columns-autofill {
    grid-template-columns: repeat(auto-fill, minmax(100px, 1fr));
}

----HTML----

<div class="grid container grid-template-columns-autofill">
    <div>1</div>
    <div>2</div>
    <div>3</div>
</div>

OUTPUT

1
2
3

auto fit in minmax

Fits our item in extra space.

CODE

----CSS----

.grid-template-columns-autofit {

----HTML----
