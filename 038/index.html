Object

The Object class represents one of JavaScript's data types. It is used to store various keyed collections and more complex entities. Objects can be created using the Object() constructor or the object initializer / literal syntax.

Example

delete operator

The JavaScript delete operator removes a property from an object; if no more references to the same property are held, it is eventually released automatically.

Example

Object Methods

Object.assign()

The Object.assign() method copies all enumerable own properties from one or more source objects to a target object. It returns the target object.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> let target = { a: 1, b: 2 };
undefined
> let source = { b: 4, c: 5 };
undefined
> Object.assign(target, source);
{ a: 1, b: 4, c: 5 }
> target; // TARGET Object
{ a: 1, b: 4, c: 5 }
> source; // SOURCE Object
{ b: 4, c: 5 }

Object.create()

The Object.create() method creates a new object, using an existing object as the prototype of the newly created object.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> let person = { name: 'Balu Vadivel' };
undefined
> let me = Object.create(person);
undefined
> me.name = 'Arya Stark'; // inherited properties can be overwritten
'Arya Stark'
> me.isFunny = 'false'; // "isFunny" is a property set on "me", but not on "person"
'false'
> me;
{ name: 'Arya Stark', isFunny: 'false' }

Object.freeze()

The Object.freeze() method freezes an object. A frozen object can no longer be changed; freezing an object prevents new properties from being added to it, existing properties from being removed, prevents changing the enumerability, configurability, or writability of existing properties, and prevents the values of existing properties from being changed.

Example - with freeze 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
.editor
// Entering editor mode (Ctrl+D to finish, Ctrl+C to cancel)
const objWF = { prop: 10, prop1 : 20  };
Object.freeze(objWF); // Freezes the object "objWF"
delete objWF.prop; // won't delete prop
objWF.prop1 = 22;  // won't change prop1
objWF.prop2 = 30;  // won't add prop2
console.log(objWF);
{ prop: 10, prop1: 20 }
undefined

Object.seal()

The Object.seal() method seals an object, preventing new properties from being added to it and marking all existing properties as non-configurable. Values of present properties can still be changed as long as they are writable.

Example 

balu@balu:~$ node
Welcome to Node.js v14.16.1.
Type ".help" for more information.
> .editor
// Entering editor mode (Ctrl+D to finish, Ctrl+C to cancel)
const number = { a: 1, b: 2 };
Object.seal(number);
delete number.a; // won't delete existing property
number.b = 20; // change existing property value
number.c =3; // won't add new property
console.log(number);
{ a: 1, b: 20 }
undefined
